# Wtyczka Joomla: Custom JS File

Prosta wtyczka systemowa dla Joomla, która automatycznie ładuje własny plik JavaScript.  
Umożliwia dodawanie niestandardowych skryptów bez edytowania szablonu lub plików systemowych.

## Funkcje

- Automatyczne ładowanie pliku `user.js`.
- Plik przechowywany jest w katalogu: `/media/plg_addjs/js/user.js`.
- Możliwość łatwego dopisywania własnych skryptów.
- Kompatybilność z Joomla 4 i Joomla 5.
- Brak potrzeby konfiguracji.

## Instalacja

1. Pobierz paczkę `.zip`.
2. Wejdź do panelu administracyjnego Joomla.
3. Przejdź do **Rozszerzenia → Instaluj**.
4. Wgraj paczkę.
5. Włącz wtyczkę w **Rozszerzenia → Wtyczki** (typ: System).

## **WAŻNE OSTRZEŻENIE**

**Ponowna instalacja wtyczki nadpisuje istniejący plik `user.js`.  
Przed reinstalacją skopiuj swój plik, aby uniknąć utraty kodu.**
